## Was ist Home Assistant

*Home Assistant* ist eine Plattform zur Hausautomation, die auf einem lokalen Server (z. B. Raspberry Pi, NAS oder PC) läuft und viele Smart‑Home‑Geräte zusammenführt. Ziel ist *lokale Steuerung* und *Privatsphäre*, nicht die Abhängigkeit von Cloud‑Diensten.
