* VerboseStatus (Voreinstellung: 0, Aktiviert: 1)

Dieser Eintrag muss neu angelegt werden (DWORD-Wert (32-Bit)). Bei Aktivierung werden beim Booten und Herunterfahren ausführlichere Informationen angezeigt.
